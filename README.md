# Linha do Tempo

Este projeto é um site evolutivo que demonstra o desenvolvimento de uma página web, começando pelo uso básico de HTML e avançando até a aplicação de estilos com CSS e animações com JavaScript.

## 📌 Funcionalidades

* Estrutura básica com HTML
* Estilização com CSS
* Implementação de animações
* Interatividade com JavaScript

## 🛠️ Tecnologias

* HTML
* CSS
* JavaScript

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar e aprofundar conhecimentos em desenvolvimento web, mostrando a evolução de um site desde o básico até recursos mais avançados.

## ▶️ Como usar

1. Baixe ou clone o projeto
2. Abra o arquivo `index.html` no navegador

## 📚 Observação

O projeto foi criado para fins educacionais.
